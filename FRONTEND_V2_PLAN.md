# 新前端 v2 设计方案与实施计划

> 定位：业界一流的专业作战管理工具 UI。企业级 SaaS 风格，参考飞书/Linear/Notion。

---

## 1. 设计原则

### 1.1 核心原则（UX 北极星）

| 原则 | 含义 | 反面教材 |
|------|------|----------|
| **专业克制** | 信息密度高但不拥挤，每屏只做一件事 | 28 个模块卡片平铺在首页 |
| **渐进展现** | 默认只看关键信息，详情按需展开 | 所有字段一次性全展开 |
| **一致性** | 同一操作在所有页面表现一致 | 状态颜色每页重新定义 |
| **即时反馈** | 每个操作都有视觉确认 | 操作后无提示 |
| **容错** | 破坏性操作必须二次确认，可撤销更好 | 删除只弹个 Popconfirm |

### 1.2 视觉规范

```
布局：
  ├─ 左侧导航栏（可折叠）200px / 64px
  ├─ 顶部工具栏 48px（面包屑 + 搜索 + 角色选择 + 通知）
  └─ 内容区 max-width: 1400px, 居中

间距体系（4px 基数）：
  xs=4  sm=8  md=16  lg=24  xl=32

色彩体系（Ant Design Token 定制）：
  主色: #1677ff → 改为更深沉的 #0050b3（专业蓝）
  成功: #52c41a
  警告: #faad14
  危险: #ff4d4f
  中性: #8c8c8c / #bfbfbf / #f0f0f0 / #fafafa

状态色（全局统一定义一次）：
  待响应 → gold     处理中 → blue
  进行中 → cyan     已解决 → green
  已关闭 → default  高 → red
  中 → orange       低 → blue

字体：
  标题: -apple-system, "Segoe UI", Roboto, sans-serif
  正文: 同上，14px，行高 1.6
  代码/数字: "SF Mono", "JetBrains Mono", monospace

阴影层级：
  level-0: none
  level-1: 0 1px 2px rgba(0,0,0,0.03)     ← 卡片
  level-2: 0 4px 12px rgba(0,0,0,0.08)     ← 弹出层
  level-3: 0 8px 24px rgba(0,0,0,0.12)     ← Modal

圆角：6px（按钮/卡片/输入框统一）
```

### 1.3 交互规范

- **新建/编辑**：统一用右侧抽屉（Drawer），不用弹窗
- **删除**：Popconfirm 二次确认，显示被删对象名称
- **列表**：分页（20 条/页），排序，列显隐，筛选栏
- **状态流转**：按钮组 + 必填备注 Drawer
- **反馈**：message.success/error（顶部居中，2s 自动消失）
- **加载**：Skeleton 骨架屏（列表页），Spin（详情页）
- **空状态**：Ant Empty + 引导文案 + 快捷操作按钮
- **导航**：左侧可折叠菜单 + 面包屑 + 浏览器前进后退

### 4.4 通用实体表格（EntityTable）

所有非攻关单的 nodeType 统一用此组件，与攻关作战台共享：
- 筛选栏 + 分页表格 + 新建 Drawer + 编辑 Drawer
- 标题列点击 → 详情 Drawer（非攻关单不需要独立详情页）
- ref 字段渲染为可点击链接（跳关联页）
- 枚举字段渲染为 StatusTag
- 字段管理放在表格右上角 ⚙ 菜单里（不改列头）

## 6. 与参考前端的关键差异

| 方面 | 参考前端（测试用途） | 新前端 v2（产品级） |
|------|---------------------|---------------------|
| 导航 | 横向 Menu 26 项溢出 | 左侧可折叠分组菜单 + 图标 |
| 主题 | Ant Design 裸默认 | 定制主题 Token + 状态色统一 |
| 新建/编辑 | 行内展开所有字段 | 右侧 Drawer 表单 |
| 表格 | 无分页、无排序 | 分页 20 条、排序、筛选、列显隐 |
| 空状态 | `<p>暂无数据</p>` | Empty 组件 + 引导操作 |
| 加载 | 无提示 / Spin | Skeleton 骨架屏 |
| 状态色 | 每页自行定义 | 全局 constants 统一定义 |
| 字段管理 | 列头 5 个小链接 | 表格右上角 ⚙ 设置菜单 |
| 图谱 | 固定 800x600 SVG | 自适应 + 平移缩放 |
| 响应式 | 无 | 移动端折叠侧边栏 |
| 图标 | 几乎不用 | 每个菜单/操作配图标 |
